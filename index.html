Funcionalidades

Conheça algumas das funcionalidades e vantagens de utilizar o OpenData Processor em sua Instituição:

Interface Administrativa

Através da interface administrativa é possível gerenciar organizações, grupos, conjuntos de dados, recursos de dados e executar as tarefas de extração.

Extração via API REST

Permite a extração de dados a partir de uma API REST com os seguintes mecanismos de autenticação: Nenhuma, HTTP Basic, Token Simples, OAUTH ou JSON Web Token.

Extração via Banco de Dados

Permite a extração de dados a partir dos SGDBs mais utilizados no mercado como SQLite3, MySQL / MariaDB, PostgreSQL e SQL Server.

Agendamento de Extração

Agende a extração automática de acordo com o nível de atualização dos seus dados ou seja diáriamente, semanalmente, mensalmente, semestralmente ou anualmente.

Integração com o CKAN

É possível importar os dados já existentes em seu Portal de Dados Abertos e a partir daí todo o processo de atualização dos conteúdos é feita de forma automática.

Auditoria e Monitoramento

Acompanhe as ações de cadastro, edição e exclusão realizadas pelos usuários e acompanhe em tempo real o processo de extração dos dados.

Interface Adaptável

A interface do sistema se adapta a diferentes telas, de forma que o conteúdo seja melhor organizado de acordo com o dispositivo do usuário.

Software como serviço

Não é necessário possuir uma estrutura e capacitar os profissionais de TI para manter o sistema funcionando. Mantenha o foco na publicação dos dados abertos.
